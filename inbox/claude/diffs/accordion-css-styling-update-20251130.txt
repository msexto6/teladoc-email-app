ACCORDION CSS STYLING UPDATE - DIFF
====================================

Task: Update accordion visual design per client specifications

CHANGES MADE

DETAILED CSS CHANGES
--------------------

--- .form-accordion
BEFORE: border: 1px solid rgba(0, 0, 0, 0.06);
AFTER:  border: 1px solid var(--color-primary-dark); /* Plum #351F65 */

--- .accordion-item spacing
BEFORE: .accordion-item + .accordion-item { border-top: 1px solid rgba(0, 0, 0, 0.06); }
AFTER:  .accordion-item + .accordion-item {
          margin-top: 30px;
          border: 1px solid var(--color-primary-dark);
          border-radius: var(--radius-xl);
          overflow: hidden;
        }

--- .accordion-header background and text
BEFORE: background: #f7f7fb;
        color: var(--color-primary-dark);
AFTER:  background: var(--color-primary-dark); /* Plum */
        color: var(--color-white); /* White text */
        padding: 14px 50px 14px 16px; /* Added right padding for arrow */

--- .accordion-header hover
BEFORE: background: #ededf7;
AFTER:  background: #2A1952; /* Darker plum */

--- NEW: White down arrow indicator
.accordion-header::after {
    content: '';
    position: absolute;
    right: 16px;
    top: 50%;
    transform: translateY(-50%) rotate(0deg);
    width: 0;
    height: 0;
    border-left: 6px solid transparent;
    border-right: 6px solid transparent;
    border-top: 8px solid var(--color-white); /* White arrow */
    transition: transform var(--transition-fast);
}

--- NEW: Arrow rotation when expanded
.accordion-header[aria-expanded="true"]::after {
    transform: translateY(-50%) rotate(180deg);
}
