# INBOX GUIDE
This folder defines the workflow for all AI agents and human developers contributing to the Email App project.

## /tasks
Instructions for work to be done: feature requests, bug fixes, refactors.

## /analysis
Diagnostics, logs, investigations, architectural notes.

## /diffs
Exact code-level changes expressed as diffs.

## /changes
Full updated files after diffs are applied.

Workflow: analysis → tasks → diffs → changes → app/current-version
